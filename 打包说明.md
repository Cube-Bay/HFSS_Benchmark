# 打包说明

这里简单记录一下使用 auto-py-to-exe 打包的方法。

## 1. 安装

```powershell
python -m pip install auto-py-to-exe
```

运行：

```powershell
auto-py-to-exe
```

## 2. 基本选项

推荐：

```text
单文件：单文件
控制台窗口：基于窗口的隐藏控制台
```

脚本位置选择主程序，例如：

```text
hfss_benchmark.py
```

## 3. 添加图片

如果要让 exe 自带模型图片，展开：

```text
附加文件
```

添加：

```text
Source:
photo.jpg

Destination:
.
```

## 4. 对应命令示例

```powershell
pyinstaller --noconfirm --onefile --windowed --add-data "photo.jpg;." "hfss_benchmark.py"
```

## 5. 运行

打包完成后，exe 一般在：

```text
output/
```

或者：

```text
dist/
```

里面。
